# Docs Audit Artifacts

This folder contains the first-pass BetterForms docs validation artifacts.

## Files
- `published-pages-checklist.md`: published inventory keyed to `SUMMARY.md`, with content type, audit scope, priority, and first-pass status.
- `source-of-truth-crosswalk.md`: the code-to-docs crosswalk showing which repo paths should be treated as authoritative for each docs area.
- `high-risk-section-audit.md`: deeper review notes for the sections audited most closely in the first pass.
- `findings-backlog.md`: prioritized mismatch list with recommended fix order.
- `structure-review.md`: information-architecture and content-density review, including suggested moves/merges.

## Scope Notes
- This audit is keyed to the published navigation, not the raw filesystem.
- Placeholder cookbook pages are tracked as published content gaps, not validated product reference.
- Some helper-file/FileMaker-side behavior may still need product-owner confirmation where it is not fully visible from the app and VFG repos.
