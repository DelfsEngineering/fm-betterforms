#Action: clipboard

Allows interaction with the clipboard

| Key | Description |
| :--- | :--- |
| action | clipboard |
| options.set | `string` that you want to be placed on the clipboard |



```
// action  object for 'cllipboard'
[
  {
    "action": "clipboard",
    "options": {
      "set": "This is some sample copy I want to copy."
    }
  }
]
```
