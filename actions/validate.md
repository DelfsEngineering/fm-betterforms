# Action: validate

Runs the validation on the form. This is useful for buttons that run Save routines.

| Key | Description |
| :--- | :--- |
| action | validate |
| options | no supported options ... yet |

```
// action  object for 'cllipboard'
[
 {
  "action": "validate",
  "options": {
  }
}
]
```

If validation fails, no further actions are processed.
