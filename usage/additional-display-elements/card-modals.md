# Card Modals

Card Modals display other form pages as their contents. 

Card Modals are opened with an `showCardModal` action.

```javascript
{
    "action": "showCardModal",
    "options": {
        "pivotX": 1,
        "width": "40%",
        "slug": "some_picker_card"// note: slug names must be URL comaptible
    }
}
```
